## Installation

1. Clone this repository

    ```sh
    git clone https://github.com/jfang324/TerminalWebsite.git
    ```

2. Install NPM packages

    ```sh
    npm install
    ```

3. Run with Vite if you just want to check it out

    ```sh
    npm run dev
    ```

4. Or build a static page
    ```sh
    npm run build
    ```
